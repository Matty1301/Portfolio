Belfry

This was a prototype I worked on that explored the concept of a 2d game world which exists within a 3d environment.

I was responsible for developing many of the game engine features such as adding support for quaternions and developing a bespoke behaviour tree system.

Implementing a complete behaviour tree system presented significant technical challenges, however the system provided much greater flexibility over the use of state machines and ultimately reduced the overall workload of the team.

Frontier

This is a mobile game I developed as a personal project using Unity and published to the Google Play Store.

For the game to be supported by a wide range of devices it required me to research and implement a number of features such as fixed input, touch screen controls and dynamic scaling to ensure a consistent aspect ratio.

The game also features cloud saves and utilizes the Google Play social platform to include a global leaderboard.
